== Description ==
Subscribe to Download Lite helps you grow your email list by offering a downloadable file after signup. Add the form to any page, collect subscriber details, and automatically send the download link by email.

Use it for lead magnets, free PDFs, checklists, ebooks, resource packs, templates, or any other downloadable freebie you want to send after subscription. If you do not want to offer a file, you can also use it as a simple email signup form.

This plugin is built for site owners who want a straightforward "email before download" workflow without building a custom funnel from scratch.

= Best For =

* Delivering a free download after email signup
* Collecting leads for newsletters and email marketing
* Offering ebooks, guides, templates, PDFs, and bonus resources
* Gating downloadable content behind a signup form

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/subscribe-to-download-lite` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress.
1. Open `Subscribe to Download Lite` from the WordPress admin menu.
1. Configure the form, email settings, and downloadable file.
1. Copy the shortcode and place it on any page or post.
1. Test the signup flow with your own email address.

== Frequently Asked Questions ==

= What does this plugin do? =

It lets you collect an email signup before giving access to a downloadable file. After the user subscribes, the plugin sends the download link by email.

= I don't have a file to offer. Can I still use it as a subscription form? =

Yes, if you don't provide any downloadable file in the plugin settings, then it will just act as the normal subscription form.

= How does the subscriber receive the file? =

The subscriber will receive the link of the downloadable file in the email through which subscriber will be able to download.

= Can I protect the download link from direct sharing? =

Yes. The plugin includes an encrypted download link option so the link is only valid for the intended subscriber flow.

= What happens when I turn on Encrypt Download Link? =

When this option is turned on in the general settings, our plugin will send the encrypted download link instead of the direct file link for download and our plugin will use the browser cookies to verify if the link has been clicked from the same browser or not from which the subscription has been done. If the link has been clicked from different browser then it will just open the home page which will prevent the direct link sharing. And also it will help to track the download status for the specific subscriber.

= Why does the download link open the home page instead of the file? =

That usually happens when Encrypt Download Link is enabled and the link is opened outside the original subscriber/browser context. Please check the FAQ above for the detail of this feature.
